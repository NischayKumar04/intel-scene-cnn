# Intel Scene Classification using CNN and Transfer Learning

This project classifies natural scene images into 6 categories:

- buildings
- forest
- glacier
- mountain
- sea
- street

The project starts with a custom CNN and then improves performance using MobileNetV2 transfer learning.
